## 一、设计分析与对比

### 1.1 参考网站分析（互动电影创作台）

| 维度 | 特点 | 可借鉴点 |
|------|------|---------|
| **布局** | 三栏布局，中间为主工作区 | 场景列表居中，操作便捷 |
| **色彩** | 深色主题，橙色强调 | 状态图标清晰 |
| **信息密度** | 高密度，专业工具风格 | 状态一目了然 |
| **交互** | 右侧面板切换标签 | 属性/资产/事件分区 |

### 1.2 现有前端分析（Memora Connect）

| 维度 | 当前状态 | 待优化点 |
|------|---------|---------|
| **布局** | 三栏布局 | 中间区域利用率低 |
| **色彩** | 浅色主题，蓝紫色强调 | 缺少层次感 |
| **信息密度** | 中等 | 场景信息展示不完整 |
| **交互** | 基础标签切换 | 缺少场景详情面板 |

## 二、设计原则

### 2.1 简约静谧风格（Soft Minimalism）

> **核心理念**: 减少视觉噪音，让内容成为焦点

**设计关键词**:
- 🌿 **柔和** - 使用低饱和度色彩，避免刺眼
- 🍃 **留白** - 充足的呼吸空间
- 🪨 **稳重** - 稳定的布局结构
- 🌙 **静谧** - 平和的视觉体验

### 2.2 设计约束

| 约束 | 说明 |
|------|------|
| ❌ 不使用 | 渐变背景、发光效果、强烈阴影 |
| ✅ 使用 | 平坦色彩、微弱阴影、清晰边框 |
| ❌ 避免 | 信息过载、复杂动画、花哨装饰 |
| ✅ 追求 | 清晰层次、一致间距、优雅排版 |

## 三、色彩系统

### 3.1 主色调

```css
:root {
  /* 主色 - 静谧紫 */
  --primary: #7C6A9E;
  --primary-light: #9B8CB8;
  --primary-dark: #5D4E7A;
  --primary-bg: #F5F3F8;

  /* 中性色 - 温暖灰 */
  --gray-50: #FAFAFA;
  --gray-100: #F5F5F5;
  --gray-200: #EEEEEE;
  --gray-300: #E0E0E0;
  --gray-400: #BDBDBD;
  --gray-500: #9E9E9E;
  --gray-600: #757575;
  --gray-700: #616161;
  --gray-800: #424242;
  --gray-900: #212121;

  /* 功能色 - 柔和 */
  --success: #81C784;
  --success-bg: #E8F5E9;
  --warning: #FFB74D;
  --warning-bg: #FFF3E0;
  --error: #E57373;
  --error-bg: #FFEBEE;
  --info: #64B5F6;
  --info-bg: #E3F2FD;

  /* 背景色 */
  --bg-primary: #FFFFFF;
  --bg-secondary: #FAFAFA;
  --bg-tertiary: #F5F5F5;

  /* 边框色 */
  --border-light: #E8E8E8;
  --border-medium: #D0D0D0;

  /* 文字色 */
  --text-primary: #333333;
  --text-secondary: #666666;
  --text-tertiary: #999999;
}
```

### 3.2 色彩应用

| 元素 | 颜色 | 说明 |
|------|------|------|
| 主按钮 | `--primary` | 静谧紫 |
| 成功状态 | `--success` | 柔和绿 |
| 进行中 | `--info` | 柔和蓝 |
| 未开始 | `--gray-400` | 浅灰 |
| 锁定状态 | `--gray-300` | 更浅灰 |
| 背景 | `--bg-primary` | 纯白 |
| 卡片 | `--bg-secondary` | 微灰 |
| 分割线 | `--border-light` | 浅灰 |
